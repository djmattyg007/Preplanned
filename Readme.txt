This is a BLT friendly version of the Mod "Preplanned" by
ThatGuyFromBreakingBad. The original can be found at
https://github.com/thatguyfrombb/Preplanned/

To install this mod, drop the folder "preplanned" into your "mods"
folder.

In order to use this mod, you need to bind the keys for "Save" and
"Load". You can do this by going to "Options" > "Mod Keybinds" while
in the game.

Modification done by Reddit user /u/DerFelix to make it compatible with
the BLT hook. Further modifications made by djmattyg007 to tidy up the
code.
